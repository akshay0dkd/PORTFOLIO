# **App Name**: Akshay's DevFolio

## Core Features:

- Hero & Introduction Display: Showcase Akshay's name, profile photo placeholder, a compelling tagline, a short introduction, and clear calls-to-action for viewing projects, downloading the resume, and contacting.
- Comprehensive About Section: Present detailed information regarding Akshay's educational background, an exhaustive list of technical skills (languages, concepts, tools), and personal interests in an organized, scrollable layout.
- Interactive Project Gallery: Display project cards with image placeholders, descriptive text including tech stack and highlights, and direct links to GitHub repositories or live demonstrations, incorporating subtle hover effects.
- Resume Download & Key Highlights: Provide a prominent 'Download Resume' button along with a summary of key achievements in DSA, Java development, and real-world project experience.
- Functional Contact Form: Implement a fully operational contact form enabling visitors to send messages directly. This includes fields for name, email, and message, with backend submission for message handling.
- Dynamic Dark/Light Mode Toggle: Allow users to seamlessly switch between a professionally designed dark theme and an equivalent light theme to cater to personal viewing preferences.
- Responsive Layout & Smooth Navigation: Ensure the website is fully responsive across all devices with a mobile-first design approach, featuring smooth-scrolling navigation between sections and a clear footer.

## Style Guidelines:

- Color Scheme: Designed primarily for a dark mode experience with a future light mode toggle. The concepts informing this palette are 'professionalism', 'technology', and 'depth'.
- Primary color: A vibrant, deep blue (#557AF2), selected to evoke confidence and highlight interactive elements against a darker background.
- Background color: A rich, desaturated dark navy (#1B2330), creating a sophisticated and professional base for the application.
- Accent color: A striking cyan-aqua (#0DCCE5), analogous to the primary color, used for calls-to-action and key distinctions to create strong contrast and draw attention.
- Headline font: 'Space Grotesk', a proportional sans-serif, to convey a modern, tech-forward, and clean aesthetic. Body font: 'Inter', a grotesque-style sans-serif, chosen for its readability and neutral appearance, perfectly complementing the headlines.
- Utilize modern, minimalist SVG icons for social media links (LinkedIn, GitHub) and navigation elements, maintaining a clean and professional look.
- Implement a mobile-first, fluid grid system with ample whitespace. Content should be clearly separated into distinct, scrollable sections with responsive image handling to ensure optimal viewing on all screen sizes.
- Incorporate subtle scroll animations for content entry, smooth scrolling between sections, and tasteful hover effects on project cards and interactive elements for an engaging user experience.
